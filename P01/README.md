
# P01: Kanban & Compte de Github

## Breu descripció

Aquest producte combina dues eines essencials per al treball professional dins la consultora EverPia:

- La gestió visual de tasques amb metodologia Kanban mitjançant Microsoft Planner.
- La creació i ús d’un repositori personal a GitHub, que serà el vostre espai de treball individual i de documentació tècnica.

Amb aquestes dues parts aconseguirem que cada grup i cada alumne tingui un sistema d’organització clar, transparent i professional, igual que qualsevol equip real del sector IT.

## Product backlog

## Objectius específics de la tasca / Finalitat de la tasca

### Objectiu del producte

- Garantir que cada grup disposa d’una eina de coordinació col·laborativa (Planner).
- Garantir que cada alumne té un espai individual de treball i documentació (GitHub).
- Reforçar el compromís amb la metodologia professional de la consultora EverPia: transparència, organització i traçabilitat del treball.

---

## Part 1: Tauler Kanban (Microsoft Planner)

Cada grup ha de crear un tauler de tipus Kanban a Microsoft Planner.

Al tauler hi ha d’haver com a mínim les següents columnes:

- **Backlog** (llista de totes les tasques pendents del projecte).
- **En curs** (tasques en desenvolupament).
- **En revisió** (tasques pendents de validació).
- **Fet** (tasques completades).

Requisits addicionals:

- El Backlog ha d’estar creat des del principi amb totes les tasques assignades al projecte (T01, T02, …, T10).
- Caldrà anar movent les tasques en funció del progrés del grup.

👉 **Lliurable:** Enllaç públic o compartit al vostre tauler Kanban.

---

## Part 2: Compte i repositori GitHub (individual)

Cada alumne ha de:

- Crear un compte personal de GitHub (si no en té).
- Associar-lo amb el correu de l’escola.
- Crear el repositori de presentació personal.
- Escriure la vostra presentació al fitxer `README.md`.

👉 **Lliurable:** Enllaç al vostre repositori personal de GitHub.
